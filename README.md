# Climartha – Smart Weather Dashboard (Java Swing)

Climartha is a modern and visually rich Java Swing Weather Dashboard that provides weather information for major Indian cities and states. It features a smooth UI, gradient backgrounds, custom weather cards, and an interactive temperature–humidity trend graph with hover tooltips.

# Features
1] Search any city/state from the predefined dataset.
2] Displays Temperature,Humidity, and Condition
3] Dynamic Climate Trend Graph (Temp + Humidity)
4] Hover tooltips for daily climate values
5] Gradient UI, emojis, smooth rendering, responsive layout
6] Built fully using Java Swing + AWT

# Tech Stack
1] Java 8+
2] Swing & AWT
3] Graphics2D Rendering
4] HashMap (Weather Data)
5] Randomized Trend Simulation

# How to Run
1] javac ClimarthaDashboard.java
2] java ClimarthaDashboard
