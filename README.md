# Climartha – Smart Weather Dashboard (Java Swing)

Climartha is a modern and visually rich Java Swing Weather Dashboard that provides weather information for major Indian cities and states. It features a smooth UI, gradient backgrounds, custom weather cards, and an interactive temperature–humidity trend graph with hover tooltips.

# Features
Search any city/state from the predefined dataset
Displays Temperature,Humidity, and Condition
Dynamic Climate Trend Graph (Temp + Humidity)
 Hover tooltips for daily climate values
 Gradient UI, emojis, smooth rendering, responsive layout
 Built fully using Java Swing + AWT

# Tech Stack
Java 8+
Swing & AWT
Graphics2D Rendering
HashMap (Weather Data)
Randomized Trend Simulation

# How to Run
javac ClimarthaDashboard.java
java ClimarthaDashboard
